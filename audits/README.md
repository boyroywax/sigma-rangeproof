# audits/

Completed external audit reports go here, one file per engagement, named by date
and auditor, e.g. `2026-07-acme.md`.

Use the sign-off template in the
[Audit and assurance](https://boyroywax.github.io/sigma-rangeproof/audit/) docs
page. Each report is bound to the specific commit it reviewed; changes after that
commit are outside its scope.
